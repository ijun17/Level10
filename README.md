# 핵심 기능

## 스킬 제작

- 코딩으로 스킬을 만들 수 있습니다.
- 필요한 MP와 쿨타임을 자동으로 평가해줍니다. 

## 몬스터와 전투

- 한 스테이지에는 하나의 몬스터가 존재합니다. 
- 몬스터를 물리치면 레벨이 상승합니다. 

## 멀티 플레이

- 다른 플레이어와 PVP를 할 수 있습니다.

## 몬스터 VS

- 몬스터와 몬스터를 싸우게 합니다.
- 레벨 제한없이 사용할 수 있습니다.
